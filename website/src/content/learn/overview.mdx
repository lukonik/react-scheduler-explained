# Overview

Welcome! this project is dedicated to teach you about React Scheduler. If you don't know what React Scheduler is thats why
this project exists in the first place. but just to give you a glance the React Scheduler is what let's React to be so performant and fast,
it allow React to have time slicing capabilities stop and resume the work. Basically it is a task manager where tasks are registered and they are
executed sequentally and efficiently without sacrificing Browser work.

The project is two parter the first part is the explaining part where the React Scheduler is explained it is under /learn
section and the second one is the react scheduler package, which is the copy of the React scheduler source code and published
you can see it under /api section [link](/react-scheduler-explained/api/introduction). I highlhy suggest you to read the learn part which is the major part of this project
and then check the package itself.

The original source code of the React Scheduler is in [here](https://github.com/facebook/react/tree/v19.2.4/packages/scheduler)
The copy of this scheduler in this project is [here](https://github.com/lukonik/react-scheduler-explained/tree/main/src)
For easier understanding of the functionaltiy of Scheduler, the code is stripped and removed the parts when it is purely For
debugging and testing purposes. But the main functionality is exactly as it is in the original code.
The vesion of the React that explains Scheduler is `19.2.4`. this learning part should be still relevant for future versions as well, I doubt it will change significantly, if it will I'll try to catch it on with it.

If you want to see the react scheduler in action check the showcase where you will find how powerful it is instead of raw executing functions, here is the link [showcase](/react-scheduler-explained/showcase)
